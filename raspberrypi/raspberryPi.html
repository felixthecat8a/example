Installing an OS into a Raspberry Pi

  1. Download and install the official Raspberry Pi Imager from the Raspberry Pi website.
  2. Insert a microSD card in a microSD card reader into your computer's SD slot.
  3. Open the Raspberry Pi Imager.
    1. Choose the "Operating System" (Raspberry Pi OS (32-bit)).
    2. Choose the "Storage" (SDXC Card).
  4. Click on the gear icon to open the Advanced Options.
    • Set the hostname as raspberrypi.local
    • Enable SSH
    • Set the username (pi) and password (raspberry)
    • Select "Configure wireless LAN" (optional)
    • Click "Save".
  5. Click "Write" to image the microSD card.
    • Alternatively, you can download and install balenaEtcher.
    • Then, download the Raspberry Pi OS from the Raspberry Pi website.
    • Click on "Flash from file" to select the Raspberry Pi OS.
    • Click on "Select target" to select the miniSD card, and click on "Flash!"
    • Remove and reinsert the miniSD card to open the "Boot" folder.
    • Create a text file with the title "ssh" on your desktop.
    • Use "Save As" to save the file in the "Boot" folder on the SD card with no extension.
  6. Insert the card in the slot on the Raspberry Pi and turn the single-board computer on.

Connecting to a Raspberry Pi via Ethernet Cable

  1. Download and install PuTTY.
  2. Connect an RJ45 cable from the Raspbery Pi to your computer.
  3. Open Putty, type in the Hostname (raspberrypi.local), and click "Open".
    • A Putty terminal will then open with a prompt to enter a username then a password.
  4. In the command promt, type "sudo raspi-config". This will open a new window.
    1. Select "interface options"
    2. Select "enable VNC", then select "yes".
  5. Set the resolution for the VNC Viewer.
    1. From the main menu, go to "display options".
    2. Go to "VNC resolution", and choose 1280x720.
    3. Select "Finish", and allow the Raspberry Pi to reboot.
  6. Download and install VNC Viewer.
    • You will need to create an account with VNC Viewer.
  7. Type in the Hostname (raspberrypi.local), and hit enter
  8. Enter the username (pi) and password (raspberry), and click "ok".

You should be able to see the Linux desktop on your computer monitor.

Update Raspian

Raspian is the Raspberry Pi Operating System

  1. Clicking on the Raspberry Pi logo on the top-left corner opens the Applications Menu.
  2. Open a terminal in the desktop.
    • Click on the Raspberry Pi logo then Accessories >> Terminal
  3. Update and upgrade Raspian.
    • sudo apt-get update
    • sudo apt-get upgrade

Raspian is installed with Python 3 and Thonny.
