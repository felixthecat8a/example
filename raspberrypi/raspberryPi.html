Installing an OS into a Raspberry Pi

  1. Download and install the official Raspberry Pi Imager from the Raspberry Pi website.
  2. Insert a microSD card in a microSD card reader into your computer's SD slot and format it. This will delete everything on the SD card.
  3. Open the Raspberry Pi Imager.
    1. Choose the "Raspberry Pi" device.
    2. Choose the "Operating System" (Raspberry Pi OS (64-bit)).
    3. Choose the "Storage" (SDXC Card).
    4. Click on "Next".
  4. On the OS customisation settings pop-up, click on "Edit Settings".
    • Under the "General" tab, set the hostname as raspberrypi.local
    • Set the username (pi) and password (raspberry)
    • Select "Configure wireless LAN" (optional: this allows WiFi connection)
    • Set the "Time Zone" and "Keyboard Layout".
    • Under the "Services" tab, select "Enable SSH".
    • Click "Save", then "Yes" to proceed to image the microSD card.
  5. Insert the card in the slot on the Raspberry Pi and turn the single-board computer on.

Connecting to a Raspberry Pi

  1. Open PowerShell and enter the command ssh [username]@[hostname].local.
    • ssh pi@raspberrypi.local
    • Enter the password ("raspberry") when prompted
  2. The command prompt will then show pi@raspberrypi:~ $.
  3. In the command prompt, type "sudo raspi-config". This will open a new window.
    1. Select "interface options"
    2. Select "VNC", then select "yes".
  4. Set the resolution for the VNC Viewer.
    1. From the main menu, go to "display options".
    2. Go to "VNC resolution", and choose 1280x720.
    3. Select "Finish", and allow the Raspberry Pi to reboot.
  5. Download and install VNC Viewer.
    • You will need to create an account with VNC Viewer.
  6. Type in the Hostname (raspberrypi.local), and hit enter
  7. Enter the username (pi) and password (raspberry), and click "ok".

You should be able to see the Linux desktop on your computer monitor.

Update the Raspberry Pi Operating System

Raspberry Pi OS is a Debian-based Linux distribution tailored specifically for Raspberry Pi hardware.

  1. Clicking on the Raspberry Pi logo on the top-left corner opens the Applications Menu.
  2. Open a terminal in the desktop.
    • Click on the Raspberry Pi logo then Accessories >> Terminal
  3. Update and upgrade Raspbian.
    • sudo apt-get update
    • sudo apt-get upgrade
